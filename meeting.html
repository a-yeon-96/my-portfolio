Meeting 프로젝트 소개

소개팅 사이트 개발

프로젝트 기간 2020년 7월 9일 ~ 2020년 7월 17일
활용 Tools Eclipse, Oracle, Java, Js
간단한 소개글 소개팅 사이트를 제작한 세미 프로젝트이며, Eclipse을 이용한 사이트입니다.

💑 소개팅 사이트 제작

👫 투입인원 : 4명

🙋‍♀️주요역할 및 담당 : 팀원



1. 로그인(상태 유지)


- JSP의 Form 기능과 input을 이용하여 아이디, 비밀번호 입력기능
- sendRedirect & getRequestDispatcher를 이용하여 페이지 이동
- 세션을 이용한 로그인 상태유지 기능 구현

2. 회원가입


- JSP의 Form 기능과 input을 이용하여 아이디, 비밀번호, 닉네임, 나이, 성별 입력기능
- MultipartRequest을 이용하여 이미지 파일 삽입 기능 구현

3. 내정보 확인


- 아이디. 닉네임, 나이, 성별, 사진 확인 가능
- 세션 유지하면서 아이디의 값을 DB에 들고가서 아이디가 같다면 DB에서 정보 가져오기
- 가져온 데이터를 보여주기 위해 StringBuilder을 이용하여 구현

4. 회원탈퇴


- invalidate()을 이용하여 세션 무효화 구현
- 아이디가Foreign key이므로 함께 삭제 기능 구현
- if문으로 부모테이블/ 부모테이블&자식테이블이 삭제되었다면 회원탈퇴 확인 구현

🔧 개발도구


- DB설계 : Oracle SQL Developer
- IDE 도구 : Eclipse
- 웹 서버 : Apache Tomcat 9.0
- 언어 : JAVA / JavaScript
