SimpleOrder 프로젝트 소개

포스와 키오스크 개발

프로젝트 기간 2020년 8월 17일 ~ 2020년 10월 14일
활용 Tools Spring, MySQL, Git, GitHub, Java, Js, Mybatis
프로젝트 GitHub https://github.com/loveiee1212/simpleOrder
간단한 소개글 포스와 키오스크를 제작한 파이널 프로제트이며, Spring을 이용하여 웹페이지를 제작한 SimpleOrder라는 플랫폼입니다. 기존의 포스, 키오스크와는 다른 차별화된 기능을 제작하였으며, 사용하기 쉽습니다. QR코드를 통하여 모바일로 사용할 수 있다는 장점입니다.

📱 [SimpleOrder] 포스와 키오스크 개발

👫 투입인원 : 6명

🙋‍♀️주요역할 및 담당 : 팀원 - 프론트엔드&CSS



1. 메인 페이지 제작


- visual studio을 사용하여 기본 베이스 틀을 지정한 후 Spring의 JSP로 제작
- BootStrap 3 활용

2. 회원가입 페이지 제작 (주소 API 구현)


- JSP의 Form 기능과 input을 이용하여 이메일, 비밀번호, 사업자번호, 상호명, 주소 등 입력기능
- 주소 API를 구현하여 주소 입력 가능
- 이메일 정규식 구현 / 비밀번호 정규식 구현
- 사업자번호 & 연락처 무조건 숫자만 입력 등 가능하게 구현
- JavaScript 이용하여 null&공백 허용하지 않도록 구현

3. 로그인


- JSP의 Form 기능과 input을 이용하여 이메일, 비밀번호 입력기능
- 이메일 정규식 구현 / 비밀번호 정규식 구현
- JavaScript 이용하여 null&공백 허용하지 않도록 구현

4. 키오스크 화면 제작 (모바일)


- 모바일 기기에 맞춤 제작

5. 모든 페이지 디자인


- 전체적인 페이지 CSS 제작

6. 유효성 검사


- 전체적인 페이지 유효성 검사 구현

🔧 개발도구


- DB 모델링: eXERD / Excel
- DB 설계: My SQL
- 웹 서버: Apache TomCat v9.0
- 주요 라이브러리: Lombok Library / maven Library / MyBatis
- 프레임워크: Spring
- 언어: JAVA / JavaScript / HTML
- 프론트엔드: jQuery / BootStrap 3 / CSS
- 버전 관리: Github
